The Inspiration: Building for Michael

It started as a simple wish. I wanted my son, Michael—a sophomore in high school—to experience the pure joy of computer-generated music composition. But standard code editors, syntax rules, and live-coding systems like Strudel can have steep learning curves for beginners.

I asked: *What if he could conduct a band just by speaking to it?* What if saying "Give me a bass drum" instantly started a loop, and saying "Make it faster" or "Drop the bass" triggered transitions in real time, with the system writing clean code behind the scenes?

Inspired by the classic text-adventure games of the 80s and 90s, where a limited vocabulary of verbs and nouns felt like complete creative freedom, we designed **Speak2Compose**. Michael can speak to a retro animated Clippy, and the system compiles his English commands into a valid Strudel script, ready to be copy-pasted directly into production tools.

Conversational Speech Parser

Supports natural phrase mapping. Chain commands with "and" or "then" to construct multi-instrument sequences in a single breath.

Fuzzy Typo Tolerant Search

Built-in Levenshtein Distance algorithm maps misheard words or speech slip-ups to the nearest musical command automatically.

Strudel Code Output

The QBasic IDE displays standard Javascript-based Strudel syntax. Highlight, copy, and paste directly into the main Strudel REPL.

Michael's Voice Command Spellbook

Rhythmic Beats

  • "give me a bass drum" or "add a kick" -> adds 4/4 bass drum loop
  • "add a snare" -> schedules snare hits on beats 2 & 4
  • "give me hi hats" -> starts active eighth-note hi-hat patterns
  • "add a clap" -> triggers synthetic claps

Melodic Synthesizers

  • "give me a bass line" -> plays pumping sub bass riffs
  • "add a synth" or "play notes" -> starts a bright lead melody
  • "play chords" -> triggers atmospheric background synthesizer pads
  • "ring bells" -> adds chime accents
  • "play strings" -> layers soft string sweeps

DJ Performance flourishes

  • "drop the bass" -> boosts kick gain and plays sub-frequency drop
  • "build the drums" -> plays a rapid 8-step snare roll riser
  • "filter sweep" -> performs highpass frequency filter modulation
  • "make it faster" / "slow it down" -> adjusts tempo by 15 BPM
  • "add echo" -> adds stereo delay echo filters
