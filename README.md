 *****************************************************************************
 *                          -  RELEASE DETAILS  -                            *
 *****************************************************************************
   Michael and his Papa bring you the ultimate offline live-coding synth
   workstation! Driven by local HTML5 speech recognition, it generates 12
   voices of synthesizer code inside a simulated DOS QBasic editor, fully 
   integrated with a real Webamp module and a helpful animated Clippy agent.

   No internet connection required. No tracking. 100% server-free!

 *****************************************************************************
 *                         -  HOW TO INSTALL & RUN  -                        *
 *****************************************************************************
   1.) Drag & drop files onto your favorite browser (Chrome/Edge recommended).
   2.) Grant microphone access when prompted.
   3.) Click on the animated Clippy agent in the bottom-right corner.
   4.) State your commands (e.g. "Add a kick and give me hi hats").
   5.) Hit the PLAY button in Webamp to start coding!

 *****************************************************************************
 *                        -  MIC VERBS & SPELLBOOK  -                         *
 *****************************************************************************
   Say these phrases to control the music (chain multiple with "AND" or "THEN"):

   [CHANNELS]
   - "give me a bass drum" / "add a kick" ---> plays s("kick*4")
   - "add a snare" --------------------------> plays s("~ snare")
   - "give me hi hats" / "add a hat" --------> plays s("hat*8")
   - "add a clap" ---------------------------> plays s("~ clap")
   - "give me a bass line" ------------------> plays note("c2 e2 g2")
   - "add a synth" / "play notes" -----------> plays note("c4 e4 g4")
   - "play chords" / "give me harmony" ------> plays note("c3 g3 a3")
   - "ring bells" ---------------------------> plays s("bell")
   - "play strings" -------------------------> plays s("strings")
   - "wind effect" / "sweep effect" ---------> triggers noise sweeps

   [MIXER CONTROLS]
   - "lower volume on [instrument]" / "quiet down [instrument]"
   - "raise volume on [instrument]" / "crank up [instrument]"
   - "mute [instrument]" / "silence [instrument]"
   - "unmute [instrument]"

   [DJ MODIFIERS]
   - "drop the bass" ------------------------> triggers low-frequency drop sweep
   - "build the drums" ----------------------> plays dynamic snare roll build
   - "filter sweep" -------------------------> sweeps lowpass frequency sweep
   - "make it faster" / "slow it down" ------> adjusts playback tempo

   * FUZZY MATCH: Typo-tolerant matching uses Levenshtein search automatically.
